# Loan Applicant Credit Risk Analysis using KNN

This Machine Learning project analyzes loan applicant data and predicts credit risk using the K-Nearest Neighbors (KNN) classification algorithm.

---

## Project Objectives

### Part-1: Data Exploration and Pre-processing

1. Load the given dataset
2. Check for null values
3. Get basic information from data
4. Describe the dataset
5. Display scatterplot between Age and Total Work Experience
6. Display box plot for Age
7. Display box plot for Cibil Score
8. Create target and feature data where target is Total Bounces Past 12 Months

---

### Part-2: Working with Model

1. Split data into training and testing sets
2. Create KNN classifier between features and target data
3. Display test score
4. Display training score
5. Print accuracy score
6. Try K values from 1 to 14
7. Display training and testing scores for all K values
